# JavaScript Learning

This repository contains my JavaScript learning journey while following the **"JS with Chai aur Code"** course.  
It’s a collection of practice code, examples, experiments, and small exercises that I’m building as I learn and explore JavaScript.

## What is this repo about?

I created this repository to:
- Learn JavaScript step by step
- Practice concepts instead of just watching tutorials
- Keep my learning organized and easy to revisit
- Track my progress over time
